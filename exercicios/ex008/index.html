Principais formatações


Negrito / Destaque

Nesta frase temos um termo em negritousando a teg b (nao semantica) ou seja declarando forma

Nesta frase temos um temos em negrito usando a teg STRONG (semantica) ou seja sem dar forma


Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag i (nao semantica) fando forma

Nesta frase, temos um termo em Ênfase usando a tag em (semantica)

Formatações adicionais em HTML

podemos criar tambem um texto marcado usando a tag MARK

para mudar a cor do marcador em algo pontual/expecifico podemos usar tag STYLE

EXEMPLOS:

TEXTO DE TESTE DE MARCADOR

se eu quiser todas os marcadores de uma so cor pra toda pagina possi usar no aba <head> </head> e uso a tag <STYLE>

texto deletado

podemos marcar um texto como excluido para indicar que ele deve ser lido mas nao considerado.

texto inserido

podemos marcar um texto como inserido para dar uma enfase e indicr que ele foi adicionado depois (voce le mais desconsidera) usando a tag <del>

tambem podemos usar a tag <u> (nao semantica, nao da sentido) para texto sublinhados ou a tag <ins> (semantica, que da sentido e nao forma)
